# SakuraMedia Frontend v0.2.3

发布日期：2026-05-07

## 概览

本次版本聚焦桌面与移动端媒体浏览筛选体验：女优列表新增排序能力，女优详情页的影片列表支持按发行年份过滤，并优化移动端关注影片卡片的封面与剧照展示策略。

## 新增与优化

- 女优列表筛选新增 `sort` 参数，支持按最近订阅、名称、影片数排序，并在桌面端与移动端共用 `ActorFilterToolbar` 交互。
- 女优详情页影片筛选新增发行年份区块，年份选项来自 `GET /actors/{actor_id}/years`，并展示为 `年份(数量)`。
- 影片列表请求新增可选 `year` 参数，女优详情页选择年份后会固定携带 `actor_id` 与单个发行年份过滤。
- 移动端关注影片卡片优先使用详情薄封面，其次回退列表薄封面；普通封面回退时改为完整包含显示，避免封面被裁切。
- 移动端关注影片剧照列表从第二张有效剧照开始展示，减少与封面区域的重复。
